# Agent Aurthur 🤖🚀

**Agent Aurthur** is a state-of-the-art, high-autonomy personal AI assistant designed to run on your own hardware. It is built for speed, privacy, and ultimate control.

> [!NOTE]
> **Lineage Credit**: Agent Aurthur is a performance-tuned distribution of [OpenClaw](https://github.com/arthur/arthur). We acknowledge and credit the OpenClaw team for the foundational code used in this project.

## 🌟 The Vision
Agent Aurthur isn't just a bot; it's a **Ghost**. It lives on your devices, consumes **zero resources** when idle, and wakes up in **under 200ms** to serve your needs across WhatsApp, Telegram, Slack, and more.

## 🚀 Key Features

- **Ghost Architecture**: 0% idle CPU burn using instant Webhook "Doorbell" triggers.
- **Titan Hybrid Mode**: Deep integration with **Mem0** (long-term memory), **Tavily** (AI search), and **PyAutoGUI** (local screen control).
- **Context Compaction**: Avoids the "JSONL trap" by using sliding window summarization to keep responses lightning-fast.
- **WhatsApp Intelligence**: Full command-driven automation directly from your favorite messenger.
- **Zero-Token Navigation**: Uses structural blueprints (`SPACE.md`) to minimize LLM token usage.

## 🛠 Strategic Optimizations

- **Sub-200ms Wake-up**: Lightning-fast reactions via Webhooks.
- **Architectural Diet**: Pruned dependencies for running on a **$5 Raspberry Pi Zero** or **ESP32**.
- **Model Tiering**: Smart routing that prioritizes **Gemini 1.5 Flash** for daily tasks.
