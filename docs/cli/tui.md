# .ravenox tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
ravenox tui
ravenox tui --url ws://127.0.0.1:18789 --token <token>
ravenox tui --session main --deliver
```
